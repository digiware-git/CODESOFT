# Simple Calculator (C++)

A basic calculator program in C++ that performs **addition, subtraction, multiplication, and division**.  
The user enters two numbers and selects an operation (`+ - * /`) to get the result.

## Features
-  Addition (`+`)
-  Subtraction (`-`)
-  Multiplication (`*`)
-  Division (`/`) with **division by zero** handling
-  Simple and beginner-friendly code

## How to Run

### 1 Compile
```bash
g++ main.cpp -o calculator
